SynAPPS on AWS

What is AWS?


  • Amazon Web Services
  • Cloud services platform
  • Compute
  • Storage
  • Content Delivery
  • Flexible
  • Scalable
  • GuildOne is only using compute, storage, and content delivery right now

Amazon EC2


  • Amazon Elastic Compute Cloud
  • Scalable Compute Instances
  • User managed

AWS Lambda


  • Compute Service for running code without a server
  • Amazon manages server runtime environment
  • Pay only for compute time used

AWS S3


  • Amazon Simple Storage Service
  • Object Storage
  • Pay for Storage and Access

AWS IAM


  • Amazon Identity and Access Management
  • Create/manage users, groups, roles, and policies
  • Grant/Deny access to AWS resources

AWS CloudFormation


  • Service for provisioning AWS resources
  • Programmatically deploy AWS environment
  • Provide template for customer environments

AWS CodeDeploy


  • Automate deployments to AWS
  • Deploys to pre-identified destination
  • Can deploy concurrently to thousands of deployment destinations

AWS Access


  • AWS Console
  • AWS CLI
  • AWS SDK
